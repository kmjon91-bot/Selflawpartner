# Blueprint: 나홀로 소송 - AI 실무 검토

## 1. 개요 (Overview)

"나홀로 소송 - AI 실무 검토"는 개인이 직접 소송을 진행할 때 작성하는 법률 서류(소장, 답변서, 준비서면 등)를 AI를 통해 사전에 검토하고, 전문가의 도움을 받을 수 있도록 연결해주는 웹 애플리케이션입니다. 복잡한 서류 작업을 돕고, 법률적 주장의 완성도를 높이는 것을 목표로 합니다.

## 2. 스타일 및 디자인 (Style & Design)

*   **테마:** 전반적으로 어두운 다크 모드(Dark Mode) 테마를 사용하며, 주황색(`--accent: #ff6a00`)을 강조색으로 사용하여 주목도를 높입니다.
*   **레이아웃:**
    *   CSS Grid를 활용한 반응형 레이아웃 (`.grid`).
    *   카드(Card) 기반의 UI 컴포넌트로 정보를 명확하게 구분.
    *   모바일 환경을 고려한 반응형 디자인 적용.
*   **타이포그래피:** 가독성이 좋은 시스템 폰트(San-serif)를 우선적으로 사용합니다.
*   **컴포넌트:**
    *   부드러운 전환 효과가 적용된 버튼.
    *   탭(Tabs)과 세그먼트(Segmented Control)를 이용해 콘텐츠를 명확히 구분.
    *   모달(Toast) 메시지를 통해 사용자에게 간결한 피드백을 제공.

## 3. 기능 명세 (Features)

### 3.1. 화면 구성
*   **시작 화면 (Splash Screen):** 서비스의 핵심 가치를 전달하고, 사용자의 시작을 유도하는 인트로 페이지.
*   **메인 앱 화면 (App):** 실제 기능이 동작하는 메인 화면으로, '대문'과 '문서 작성/업로드' 섹션으로 구성.

### 3.2. 문서 입력 방식
*   **구조화 입력:**
    *   '당사자', '청구취지', '청구원인', '증거목록'을 각각의 입력 필드에 나누어 작성.
    *   당사자(원고/피고)는 동적으로 추가 및 삭제 가능.
    *   작성된 내용을 바탕으로 하나의 최종 문서를 생성.
*   **자유 입력:** 사용자가 이미 작성해 둔 문서 전체를 텍스트 영역에 붙여넣기.
*   **파일 업로드:**
    *   `.txt`, `.pdf`, 이미지 파일(`.png`, `.jpg`) 업로드 지원.
    *   **텍스트 기반 PDF:** 브라우저 내에서 `pdf.js` 라이브러리를 사용해 텍스트를 직접 추출.
    *   **이미지 기반 PDF / 이미지 파일:** 서버의 OCR API (`/api/ocr`)를 호출하여 텍스트를 추출. (백엔드 구현 필요)

### 3.3. AI 검토
*   **규칙 기반 검토 (MVP):**
    *   '최종 문서'의 텍스트를 바탕으로 사전 정의된 규칙에 따라 기본 항목(당사자, 청구취지, 청구원인 등)의 누락 여부를 검사.
    *   선택된 '소송 단계'와 '문서 종류'에 따라 특화된 점검 항목(Addendum)을 추가로 제공.
*   **검토 결과:**
    *   점검 결과를 리포트 형태로 제공.
    *   결과 텍스트를 복사하거나 `.txt` 파일로 다운로드 가능.

### 3.4. 전문가 연결
*   **전문가 추천:** AI 검토 결과(누락/위험 신호 개수)와 문서 종류를 분석하여 '변호사', '법무사', '노무사' 등 적합한 전문가 유형을 추천.
*   **서비스 추천:** 상황에 맞는 서비스('기존 서면 검토', '전문가 서면 작성')를 추천.
*   **연결 요청:**
    *   사용자가 연락처와 추가 요청사항을 입력.
    *   AI 검토 결과, 사용자 입력 등을 포함한 요청 요약을 생성하여 클립보드에 복사. (MVP에서는 실제 전송 대신 복사 기능으로 제공)

## 4. 현재 실행 계획 (Current Plan)

1.  **프로젝트 초기화:** 사용자가 제공한 단일 HTML 코드를 기반으로 프로젝트를 재구성합니다.
2.  **파일 생성:** `index.html` 파일을 생성하고 제공된 코드를 삽입합니다.
3.  **기능 확인:** 초기화된 프로젝트가 의도대로 동작하는지(특히 페이지 전환) 확인합니다.
